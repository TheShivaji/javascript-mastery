## 📌 About

This repository represents an immersive, hands-on journey to mastering **modern JavaScript (ES6+)**. Rather than focusing on passive learning, every single folder contains production-grade challenge solutions designed to build a deep, intuitive understanding of core programming mechanics, data structure algorithms, and functional design patterns.

---

## 🗺️ Learning Roadmap

A structured, logical path to JavaScript engineering excellence.

- [x] **01. Variables** — Basic storage, scope, and variable declaration (`let`, `const`, `var`).
- [x] **02. Operators** — Arithmetic, assignment, logic, comparison, and operator precedence.
- [x] **03. If/Else Logic** — Branching, truthy/falsy values, and conditional logic.
- [x] **04. Functions** — Arrow functions, declarations, expressions, parameters, and return types.
- [x] **05. Arrays** — Memory structures, iteration, and array manipulations.
- [x] **06. Objects** — Key-value pair storage, accessors, context scope (`this`), and object methods.
- [x] **07. Loops** — Loops (`for`, `while`), loop nesting, control statements, and array iteration.
- [x] **08. Problem Solving** — Formulating solutions, data transformations, and algorithm structuring.
- [ ] **09. Strings & RegEx** — Regular expressions and advanced string mutations.
- [ ] **10. DOM & Event Handling** — Interacting with markup and dynamic page manipulation.
- [ ] **11. OOP & Classes** — Prototypal inheritance, ES6 classes, instantiation, and encapsulation.
- [ ] **12. Async JavaScript** — Event Loop, Promises, `async/await`, and concurrency models.
- [ ] **13. Modern ES6+ & Modules** — Destructuring, spread/rest, optional chaining, and modular architecture.

## 🚀 How to Run the Code

To execute any daily practice script locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/TheShivaji/javascript-mastery.git
   ```
2. Navigate into the folder:
   ```bash
   cd javascript-mastery
   ```
3. Run the JS file using Node.js:
   ```bash
   node Day-08-Coding-Challenges/challenge-01.js
   ```
