## About This Repository

This repository contains a collection of beginner-friendly JavaScript practice files organized by topic and day. Each challenge is preserved as originally written, with only light formatting and spelling fixes applied.

2. Operators and expressions
3. Conditional logic with if/else
4. Functions and reusable logic
5. Arrays and array methods
6. Objects and object methods
7. Loops and iteration
8. Coding challenges and problem solving

1. Open a terminal in this repository.
2. Run any file with Node.js:

```bash

```

3. To view the HTML landing page, open the file in your browser:

```text
html/index.html
```
